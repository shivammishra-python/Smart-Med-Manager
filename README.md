# Smart Med Intelligence System

A Python-based Command Line Interface (CLI) application developed to track medication adherence, manage vaccine/medicine registries, and monitor inventory status in real-time through the terminal.

## 🚀 Features
- *Add New Medicine/Vaccine:* Dynamic user inputs to register medications.
- *View All Reminders:* Displays active medication timelines.
- *Check Inventory Status:* Uses conditional filtering to automatically flag low stock status thresholding.
- *Exit & Save:* Clean exit parameters for data integrity.

## 🛠️ Core Concepts Used
- *Data Structures:* Python Dictionaries and Lists for modular data storage.
- *Control Flow:* Complex for loops and if-elif-else conditional structures.
- *User Interface:* Interactive terminal menu driven by continuous execution loops.
